#### 条件组合覆盖

第三个判断的外层复合条件记为（这里的 `param1`、`param2`、`param3` 都指执行完前两个 `if` 后的当前值）：

```text

Q = param1 > param3
```

```text

S = param1 <= param3 = !Q
```
